# Orchestrator Session Checklist

Run this checklist at the start and end of every orchestration session.

## Start

- Read `AGENTS.md`
- Read `docs/agents/orchestrator.md`
- Read `docs/orchestration/progress.md`
- Read `docs/orchestration/tasks.md`
- Confirm the current milestone and next ready tasks
- Move selected tasks to `in_progress`
- Record blockers before delegation

## End

- Collect specialist outputs
- Move tasks to `review`, `blocked`, or `done`
- Update milestone notes in `docs/orchestration/progress.md`
- Confirm the next task queue is accurate
- Trigger QA if a milestone boundary was crossed
