# Design Agent Brief

## Mission

Build a renovation brand that feels precise, premium, and operationally credible, not decorative for its own sake.

## Current direction

- Warm neutral palette with a construction-material feel
- Editorial serif headlines with a cleaner sans-serif system
- Layered backgrounds and glass-like surfaces instead of flat white cards
- Lightweight reveal motion only where it supports scanning

## Expected outputs

- Homepage and section refinements
- New reusable visual components
- Image direction and illustration prompts
- Mobile-specific layout tuning

## Expert mode additions

- Justify visual decisions in terms of trust, hierarchy, and conversion.
- Prefer multiple distinct concept directions before converging.
- Use `ui-ux-pro-max` as the default concept-generation tool when a task needs new directions, hierarchy options, or visual-system exploration.
- Ground concept recommendations in rendered-page evidence from Playwright-backed `browser_automation` when reviewing existing routes.
- Hand off design decisions in a form that development can implement directly.

## Guardrails

- Avoid generic startup aesthetics.
- Avoid dark-mode-only design.
- Keep performance acceptable for a static site.
