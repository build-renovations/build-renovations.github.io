Status: DONE
Owner: Content Agent
Tool mode: repo_only
MCP usage: repo_only was sufficient; microcopy derived from the implemented UX direction and bilingual site structure.

## Objective

Standardize short UI text so the site sounds practical, confident, and relevant to renovation buyers in both Ukrainian and English.

## Tone rules

- sound operational, not promotional
- ask for project context, not generic lead capture
- avoid loud promises and filler adjectives
- prefer вАЬproject discussionвАЭ, вАЬbriefвАЭ, вАЬscopeвАЭ, and вАЬstagesвАЭ over vague вАЬconsultationвАЭ language

## Trust-section framing

- Use language that signals:
  - process clarity
  - accountability
  - real examples
  - cross-trade coordination

## Avoid

- `best quality`
- `premium result`
- `free consultation`
- `leave your number and we will call back`
- any copy that sounds like a generic ad network landing page
