# Agent Output Workspace

This directory stores agent-produced artifacts, reviews, briefs, audits, and status files.

It is not part of the published Jekyll site.

## Layout

- `agents/design/`
- `agents/seo/`
- `agents/content/`
- `agents/qa/`
- `agents/performance/`
- `agents/status/`

The Jekyll development agent writes actual implementation changes in the site root.
